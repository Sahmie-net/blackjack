# Blackjack Practice Project

This is a small web-based Blackjack project I’m building as a hands-on way to practice front-end development. The goal is to make a simple, playable version of the game with a clean interface and basic game flow.

## What this project includes
- A minimal Blackjack experience in the browser
- Basic player interaction with hit and stand actions
- A lightweight setup built with HTML, CSS, and JavaScript
- Room for future improvements such as score tracking, betting, or a stronger game engine

## How to run it
Open the project in a browser by launching the main HTML file.
